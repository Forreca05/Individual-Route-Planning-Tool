\doxysubsection{Detailed Description}
Main program for loading a graph and performing various algorithms. 

This file contains the main logic for reading a graph from CSV files and running different algorithms based on the user input. The program supports both terminal-\/based interaction and batch processing modes. It performs operations based on the specified type\+: independent, restricted, or environmentally.
