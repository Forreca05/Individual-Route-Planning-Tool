\doxysubsection{Detailed Description}
Implementation of search algorithms for graph-\/based navigation. 

This file contains the main functions used to perform searches in a graph. It includes implementations for Dijkstra\textquotesingle{}s algorithm, path retrieval, edge selection, and relaxation functions for different transportation modes.
