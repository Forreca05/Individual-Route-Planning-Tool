\doxysubsection{Detailed Description}
Header file for the Restricted route algorithm. 

This header defines the functions related to the \char`\"{}restricted\char`\"{} algorithm, which computes the best route considering specific restrictions. These restrictions may include\+:
\begin{DoxyItemize}
\item Avoiding specific nodes.
\item Avoiding specific segments of the graph (edges).
\item Simultaneously avoiding nodes and edges.
\item Including specific nodes that the route must visit.
\end{DoxyItemize}

Runs the restricted pathfinding algorithm, considering avoidance and inclusion constraints. 

This function computes the shortest path between two nodes in the graph, taking into account the avoidance of specific nodes and segments, and including a specific node if required. If an inclusion node is specified, the algorithm first finds the shortest path to this node and then proceeds from that node to the destination.
