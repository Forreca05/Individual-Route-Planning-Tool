\doxysubsection{Detailed Description}
Header file for environmentally friendly route algorithm. 

This file contains function declarations for calculating and processing environmentally friendly routes that combine both driving and walking. The routes aim to minimize driving while ensuring that walking distances from parking nodes to the destination are within the user\textquotesingle{}s maximum walking time.
