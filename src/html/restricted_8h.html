void restricted ( Graph< int > &  graph)

Handles user input for restricted routes.

This function reads user input for mode, source, destination, nodes to avoid, edges to avoid, and an optional node to include in the path. It then calls the restricted pathfinding algorithm.

Parameters
graphThe graph representing the road network.

Complexity is O((V + E) log V), due to calling the pathfinding algorithm.
